# dotfiles
A repository to store aliases, code snippets and other code related things

## Setup

1. `cd; mkdir -p dev/github; cd dev/github;`
2. `git clone https://github.com/CodyCahoon/dotfiles.git`
3. `cp dotfiles/* ~; cd`
4. `sh scripts/setup.sh`
5. `source .zshrc`
